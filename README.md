# 🌐 Purav Ky — Personal Developer Portfolio

Welcome to my personal portfolio website — a modern, minimal, and responsive showcase of my skills, projects, and contact information. This site is designed to reflect my personality as a developer and designer, with clean UX, dark/light mode toggle, and elegant iconography.

---

## ✨ Features

- ⚡️ Fast, responsive design for all screen sizes
- 🌗 Dark and light theme toggle
- 🎯 Smooth transitions and hover animations
- 🛠️ Skills section with animated progress bars
- 📁 Projects with hover effects and demo links
- 📬 Contact section with clickable social icons
- 🧩 Built with clean HTML5, CSS3, and vanilla JavaScript
- 🎨 Icon system powered by Iconoir
- 💻 Fully customizable and open to contributions

---

## 🛠 Tech Stack

- **HTML5** — semantic and accessible structure
- **CSS3** — modern layout using Flexbox + Grid
- **JavaScript** — interactive components and theme handling
- **Iconoir** — clean, modern icon set via CDN
- **Google Fonts** - Source Code Pro used as the font
